# Space Survivor

A simple vampire survivors clone made in 24 hours for the Quackhacks 2025 event. The idea was to make a 3D version of vampire survivors with Galaga inspired space ships.

## Controls

W/S/Up/Down - Accelerate
Space - Break
Esc - Pause Menu
